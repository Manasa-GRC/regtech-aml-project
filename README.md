# AML Screening Dashboard 🛡️

An Anti-Money Laundering (AML) screening and transaction monitoring dashboard built with Python and Streamlit.

Built as part of my RegTech & Compliance Automation learning journey.

## What This Project Does

- Screens transactions against a sanctions watchlist
- Flags high-risk and suspicious transactions automatically
- Visualises transaction risk data using interactive Plotly charts
- Exports flagged records to CSV for compliance reporting

## Tech Stack

- Python
- Streamlit
- Plotly
- Pandas

## Why I Built This

As a Cybersecurity professional transitioning into GRC and RegTech, I wanted to apply Python practically — not just as a developer skill, but as a **compliance automation capability**. AML screening is a real RegTech use case relevant to banks and fintechs operating under FCA and FATF frameworks.
